本地安装与全局安装·

  1. npm模块本地安装与全局安装有什么区别？

    本地安装（npm install xxx）：是将模块下载到当前命令行执行所在目录；

    全局安装（npm install -g xxx）：模块将被下载安装到【全局模块目录】中。

常用命令·

查看过期插件（可更新插件）·

npm outdated

查看全局已安装·

查看全局已安装（-g 的意思是 global 全局的意思）

npm ls -g

加上层级控制显示深度：--depth 0，这样就只会查到安装的包，并不会查到包的依赖。

npm ls -g --depth 0

查看项目中已安装·

查看当前项目已安装包（项目跟目录必须有 package.json 文件）

npm ls

加上层级控制显示深度：--depth 0，这样就只会查到安装的包，并不会查到包的依赖。

npm ls --depth 0

如果只想显示生产环境依赖的包

npm ls --depth 0 --prod

只显示开发环境依赖的包

npm ls --depth 0 --dev
